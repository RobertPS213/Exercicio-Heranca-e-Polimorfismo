# 🚀 Exercícios de Java: Herança, Polimorfismo e Downcasting

Este repositório reúne exercícios práticos desenvolvidos para dominar os pilares da **Programação Orientada a Objetos (POO)** em Java. O foco principal é a aplicação de herança para reutilização de código e polimorfismo para flexibilidade do sistema.

---

## 📁 Exercícios Inclusos

### 1. 🎮 ExercicioBattleSystem (Simulação de Combate)
Uma simulação de unidades militares inspirada em mecânicas de jogos de estratégia.
* **O que faz:** Gerencia diferentes unidades (`Tanque` e `Infantaria`) em uma lista única. Cada unidade possui uma forma diferente de atacar e consome recursos específicos (combustível ou vida/fadiga).
* **Conceitos:** Polimorfismo com `@Override` e manipulação de atributos de subclasses.

### 2. 👥 ExercicioEmployee (Gestão de Pagamentos)
Sistema para cálculo de folha de pagamento de funcionários próprios e terceirizados.
* **O que faz:** Calcula o pagamento de colaboradores com base em horas e valor por hora. Para funcionários terceirizados (`OutsourcedEmployee`), o sistema aplica um bônus adicional de 110% sobre a despesa adicional.
* **Conceitos:** Herança, Polimorfismo e a palavra-chave `super` para reaproveitamento de lógica da superclasse.

### 3. 🏗️ ExercicioTaskManager (Gestão de Tarefas)
Um sistema de controle de produtividade para equipes de desenvolvimento e design.
* **O que faz:** Processa uma lista de tarefas genéricas. O programa identifica se a tarefa é de "Código" ou "Design" para executar funções exclusivas, como rodar testes unitários ou gerar protótipos no Figma.
* **Conceitos:** Downcasting seguro com o operador `instanceof`.
